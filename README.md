RTB4FREE Bidder
===============

This is the bidder component to the RTB4FREE open source DSP.  The bidder is a JAVA 11 based openRTB bidding system and built on top of Hazelcast, Postgres, Kafka, Mino/S3, Logstash, and Elastic Search.
